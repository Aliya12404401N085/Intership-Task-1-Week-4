# Clustering Techniques Report

## Objective
To understand and implement unsupervised learning clustering algorithms using the Iris dataset.

## Algorithms Used
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN

## Dataset
Iris Dataset (Scikit-learn)

## Libraries Used
- pandas
- numpy
- scikit-learn
- matplotlib
- scipy

## Results
The clustering algorithms grouped similar samples into clusters. K-Means created three clusters, Hierarchical Clustering grouped data based on similarity, and DBSCAN identified dense regions while detecting possible outliers.

## Conclusion
This task demonstrates how unsupervised learning can discover patterns in unlabeled data using different clustering techniques.
